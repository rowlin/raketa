Для запуска проекта необходимо выполнить 
```
make init
```
Для запуска тестов :
```
make tests 
```
Для запуска команды
```
make command arg='redis add key value'

make command arg='redis remove key'
```
Подробнее Makefile

Данная реализация требует рефакторинга , но требования изложенные в тз выполняет.
